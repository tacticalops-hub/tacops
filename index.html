Standoff Tactical Map

Click to add/edit a label. Draw to plan routes. Zoom with double-click or pinch. Drag when zoomed.

Keyboard Controls:
- Press Tab to navigate between elements
- When canvas is selected (blue outline):
  • Delete key to clear canvas
  • Shift + Click to add marker
- When marker is selected (blue outline):
  • Enter to edit text
  • Delete to remove marker
- Global shortcuts:
  • Ctrl + Z to undo
  • Ctrl + Y to redo
